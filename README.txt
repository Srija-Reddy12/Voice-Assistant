Jarvis Assistant (Voice-Controlled Python Project)

How to run:
1. Install Python 3.x from https://www.python.org/
2. Open terminal or command prompt.
3. Run: pip install -r requirements.txt
4. Run the assistant: python jarvis_assistant.py
5. Speak commands like:
   - What is the time
   - Open YouTube
   - Who is Elon Musk
   - Open Notepad
   - Stop

Make sure your microphone is working!
